# Milo Tap & Listen

A tiny, static toddler game for GitHub Pages.

## Audio files
Put recordings in: `assets/audio/`

Filename format:
`{Person}__{mode}__{item}__{clip}.m4a`

Examples:
- Emma__food__banana__prompt.m4a
- Dad__numbers__8__label.m4a
- Lucas__feedback__correct__v1.m4a

## Content mapping
Edit `data/content.json` to match your audio filenames.

## Run locally
Just open `index.html` in a browser, or use a simple static server.
